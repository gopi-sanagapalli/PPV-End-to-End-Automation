# Test Data (Simple)

This folder is kept simple for quick understanding.

## Files

- `desired-capabilities.json`
  - Contains only major iOS Appium desired capabilities.
  - No test flow steps are mixed here.

- `ppv-inputs.json`
  - Contains only frequently changed PPV values.
  - Keep this small (name/date/time/market/currency).

## Note

- Sample result JSON files were removed to avoid confusion.
- Keep test flow/scenario steps in spec files or separate flow files, not in desired capabilities.
